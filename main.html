Question-1: What are the different ways to select an element in the DOM?

1. document.getElementById('elementId') to select an element with a specific ID.
2. document.getElementsByClassName('className') or document.getElementsByTagName('tagName') for multiple elements.
3. document.querySelector('selector') for a single element or document.querySelectorAll('selector') for multiple elements.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Retrieves or sets the HTML content inside an element, including tags.
innerText: Retrieves or sets only the visible text content, ignoring hidden elements and respecting CSS styling.
textContent: Retrieves or sets all text content, including hidden elements, without interpreting HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you attach a single event listener to a parent element to manage events for all its child elements. This leverages event bubbling, allowing you to handle events efficiently, especially for dynamically added elements, reducing the need for multiple event listeners.

Question-4: What is event bubbling in the DOM?

Event bubbling is a behavior in the DOM where an event triggered on a child element propagates upward through its ancestor elements in the hierarchy. This allows parent elements to handle events originating from their children, enabling techniques like event delegation.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: Use document.createElement('tagName') to create a new element.
Add: Use parentElement.appendChild(newElement) or parentElement.insertBefore(newElement, referenceElement) to add the element to the DOM.
Remove: Use parentElement.removeChild(childElement) to remove an element from the DOM.
